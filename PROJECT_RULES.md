# EcoSave Development Rules

## Design

- Never redesign existing pages without permission.
- Use the existing color palette.
- Use the existing typography.
- Maintain the same spacing system.

## Technology

- HTML
- CSS
- Vanilla JavaScript only

## Architecture

- Reuse components.
- Never duplicate CSS.
- Keep code modular.

## Commit Style

- One feature per commit.
