# EcoSave Architecture

## Folder Structure

assets/
css/
js/

## Rules

Every page must reuse:

- navbar
- footer
- buttons
- cards

All CSS must be stored inside:

css/style.css

All reusable JavaScript must be stored inside:

js/script.js

Shared layout rendering is stored inside:

js/layout.js

Never duplicate CSS or JavaScript.

Every feature should be implemented in a separate commit.
